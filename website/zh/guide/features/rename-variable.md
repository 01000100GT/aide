# 智能重命名变量

命令名称: `aide.renameVariable`

使用 AI 给出几个可能的变量名建议以及解释，帮助您更快地重命名变量。

::: warning 警告

该功能需要 AI 模型支持 `function_call` 功能

:::

**使用场景：**

- 你经常为变量名苦恼，总觉得不够满意。
- 你希望 AI 能根据逻辑提供更好的变量名，并解释原因。

**使用方法：**

- 在编辑器中选择变量。
- 点击右键选择 `✨ Aide: 重命名变量`。
- 选择一个建议的变量名。
